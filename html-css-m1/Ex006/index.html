Lista Definições

HTML
Linguagem de Marcação de Hipertexto, um código que define a estrutura e o conteúdo de uma página web, como textos, imagens e links, e é interpretado pelos navegadores para que o conteúdo seja exibido corretamente.
CSS
linguagem de estilo que define a aparência de documentos escritos em linguagens de marcação como HTML, controlando aspectos visuais como cores, fontes, espaçamento, layout e animações.
JavaScript
é uma linguagem de programação que permite adicionar interatividade e dinamismo a páginas da web e aplicativos.
